선택 사항
-------------
* 처음 선택해야 하는 것은 Tcp Or Http의 선택이다
  * Tcp 서버와 Http 서버 중 Scale Out 편한 것은 Http이다
  * Http를 선택하는 것이 좋다
* 다음은 언어의 선택이다
  * 2024년 가장 핫한 서버 언어는 c#(net core)일 거 같다
  * C#을 강추한다
* 다음은 기능으로 대기 서버는 아래 기능이 가능해야 한다
  *  클라이언트 버전 체크
  *  퍼블리셔 로그인
  *  대기 번호 발급
  *  대기 완료 체크
  *  게임 서버 연계
  *  게임 서버 상태 체크
  *  운영툴을 통해 입력한 유저 유지 시키기
